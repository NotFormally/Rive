# Limites de Déploiement Vercel (Hobby Plan)

Il n'y a **aucune limite** pour travailler et modifier le code en local sur votre propre machine.

Cependant, étant donné que le site Rive est hébergé sur Vercel avec le forfait gratuit (Hobby plan), voici les limites de déploiement (mise en ligne) à garder à l'esprit :

1. **Limite par jour** : Maximum **100 déploiements par jour**.
2. **Limite par heure** : Maximum **32 déploiements par heure**. (En cas de dépassement, vous devrez attendre l'heure suivante pour pouvoir redéployer).
3. **Temps de compilation (Build time)** : Un total de **100 heures par mois** pour la construction des builds.

**En résumé :** 
Pour une utilisation normale par un développeur humain, ces limites sont extrêmement généreuses. À moins de lancer un déploiement toutes les deux minutes sans interruption, vous ne devriez jamais être bloqué. Vous pouvez donc faire toutes vos modifications sereinement !
